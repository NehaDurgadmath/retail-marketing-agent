# 🛒 Retail Marketing Intelligence Agent

An autonomous LangChain-powered AI agent that generates 
personalised marketing campaigns for retail customer segments.

## What It Does
Give it a customer segment → it automatically:
1. 🔍 Analyses buying behaviours, pain points and motivations
2. 💡 Generates 3 targeted campaign ideas with expected ROI
3. ✉️ Writes personalised email copy with Australian tone
4. 📊 Scores campaign effectiveness out of 100

## Example Output
**Segment:** Young families buying organic and baby products
**Campaign Score:** 91/100 — Recommendation: Launch ✅

## Tech Stack
Python · LangChain · OpenAI GPT-4o-mini · Multi-step Agentic Pipeline

## How to Run
```bash
pip install langchain-core langchain-openai
export OPENAI_API_KEY="your-key"
python retail_agent.py
```
